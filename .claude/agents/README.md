# Agents

Thư mục chứa các sub-agent chuyên biệt cho project.

## Cách dùng
Mỗi file `.md` là một agent với frontmatter định nghĩa vai trò và tools được phép dùng.

## Agents dự kiến
- `copywriter.md` — Viết nội dung, tagline, mô tả sản phẩm tiếng Việt
- `designer.md`   — Gợi ý màu sắc, layout, UI theo brand nước mắm truyền thống
- `seo.md`        — Tối ưu meta tags, heading structure, schema markup
