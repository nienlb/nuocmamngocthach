# Skills

Thư mục chứa các skill tái sử dụng trong project.

## Cách dùng
Mỗi file `.md` là một skill Claude có thể gọi bằng `/skill-name`.

## Skills dự kiến
- `add-section.md`    — Thêm một section mới vào index.html theo chuẩn project
- `optimize-image.md` — Checklist tối ưu ảnh trước khi thêm vào assets/
- `deploy-check.md`   — Kiểm tra trước khi deploy (links, images, meta tags)
