# Rules

Thư mục chứa các quy tắc Claude phải tuân theo khi làm việc trong project.

## Cách dùng
Mỗi file `.md` là một tập quy tắc cho một ngữ cảnh cụ thể.

## Rules dự kiến
- `coding.md`   — Quy tắc viết HTML/CSS/JS (naming convention, indentation...)
- `content.md`  — Giọng văn, thuật ngữ nước mắm, không dùng từ ngữ nào
- `brand.md`    — Màu sắc, font, hình ảnh được phép dùng
